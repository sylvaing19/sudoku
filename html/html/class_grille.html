Detailed Description

La classe Grille définit une grille de sudoku de 9*9 cases. Elle permet de modifier et/ou de consulter les valeurs contenues dans les cases, ainsi que de remplir la grille selon les règles du sudoku. Les valeurs contenues dans la grille sont des entiers compris entre 0 et 9 inclus. La valeur 0 correspond à une case vide.

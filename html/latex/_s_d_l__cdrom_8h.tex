\subsection{Detailed Description}
This is the C\+D-\/audio control A\+P\+I for Simple Direct\+Media Layer

In order to use these functions, \hyperlink{_s_d_l_8h_a8fc8d35348d7c74bad8392d776c937b8}{S\+D\+L\+\_\+\+Init()} must have been called with the S\+D\+L\+\_\+\+I\+N\+I\+T\+\_\+\+C\+D\+R\+O\+M flag. This causes S\+D\+L to scan the system for C\+D-\/\+R\+O\+M drives, and load appropriate drivers.
